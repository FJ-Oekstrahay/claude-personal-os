---
name: Google Drive path mapping
description: Local filesystem path for Google Drive files — how to interpret paths Geoff provides
type: reference
---

When Geoff provides a path like `/Users/moltyjoe/Library/CloudStorage/GoogleDrive-geoff.k.hoekstra@gmail.com/My Drive/Drone/Publishing/file.docx`, it means the file is on his Google Drive at `Drone/Publishing/file.docx`.

The Google Drive root maps to: `/Users/moltyjoe/Library/CloudStorage/GoogleDrive-geoff.k.hoekstra@gmail.com/My Drive/`
