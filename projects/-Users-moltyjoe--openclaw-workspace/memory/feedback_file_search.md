---
name: File Search Behavior
description: When a file location is unknown, ask Geoff whether to search or have him tell me
type: feedback
---

When I need to find a file and don't know its location, ask: "Want me to search for it, or can you tell me where it is?"

**Why:** Searching can be slow and noisy. Geoff often knows exactly where something is and prefers to just say.

**How to apply:** Any time a file reference is ambiguous or not found in the obvious places — don't silently search, ask first.
