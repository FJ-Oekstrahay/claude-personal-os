- Pack "archive" not "retire" — history preserved, hidden from default list
- Multi-craft sessions supported — craft_id at Flight level
- Prop tracking fully optional — explicit "not tracking" is fine, no nag
- Hours accumulate from flight duration_s (pack-level), summed for craft total — event-driven, no explicit session-end command
- JARFACE personality: dry, direct, in-character with original JARVIS. Blunt about hardware state, never deferential.
