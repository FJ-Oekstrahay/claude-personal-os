**bf_bridge.py:** New `BFBridge` class handles launch, CDP connect, tab navigation, disconnect/reconnect around serial writes. Serialized port access pattern: disconnect_bf() before every agent write, reconnect after. Tab selectors validated against real BF 10.10.0 DOM but NOT tested against a live connected FC yet.

**osd_demo.py:** Reads backup, renders 15×30 ASCII grid, asks AI for layout, validates positions in Python. Two-layer fix for OSD off-screen bug: system prompt warns col > 29 is off-screen AND Python rejects col >= 30 or row >= 15. Root cause: OSD formula is row*64+col (64-stride) but display is only 30 cols wide — AI models naively place "right side" at col 59.

**max_tokens bump:** ai.py changed 1024 → 4096. Was truncating JSON for complex sessions.
