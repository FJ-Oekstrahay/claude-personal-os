## What was built

API proxy to prevent bundled Anthropic key from shipping in the CLI binary.

**Files created:**
- `proxy/src/worker.js` — Cloudflare Worker: validates license token, enforces rate limits via KV, locks free tier to Haiku, forwards to Anthropic
- `proxy/wrangler.toml` — deploy config (KV namespace ID placeholder needs filling)
- `proxy/package.json` — wrangler dev dep
- `proxy/DEPLOY.md` — one-time setup runbook

**`cli/ai.py` changes:**
- `BUNDLED_KEY` removed; replaced with `PROXY_TOKEN_FREE` and `PROXY_TOKEN_JARFACE` (both `None` — not wired up yet)
- `_ask_proxy()` added using `urllib` (no new dep)
- `load_config()` returns `provider: 'proxy'` when proxy tokens are set
- `ask()` accepts `model` param; routes proxy tier through `_ask_proxy()`
- `_ask_anthropic()` now accepts `model` kwarg

**Docs updated:** DECISIONS.md (new proxy entry), LAUNCH_CHECKLIST.md (per-device cap items split into proxy deploy + token wiring), SPEC.md (sections 3.4, 5, 6, 7, 8)

## What's still needed before bundled tier ships

1. Create Cloudflare KV namespace, paste IDs into `wrangler.toml`
2. Generate license tokens: `python3 -c "import secrets; print('dt-free-v1-' + secrets.token_hex(16))"`
3. `wrangler secret put` for `ANTHROPIC_API_KEY`, `TOKEN_FREE`, `TOKEN_JARFACE`
4. `wrangler deploy`
5. Point `api.droneteleofpv.com` DNS at the Worker
6. Paste tokens into `cli/ai.py` `PROXY_TOKEN_FREE` / `PROXY_TOKEN_JARFACE`

Full runbook: `proxy/DEPLOY.md`
