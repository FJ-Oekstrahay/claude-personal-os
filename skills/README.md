# Skills

Invoked by Claude via the Skill tool, triggered by context rather than explicit user command. Each skill has a `when to use` guard in its SKILL.md.

| Skill | When to use |
|---|---|
| critic | Adversarial review of a plan, code, config, or architecture decision — find what fails, not what works |
| debug-agent | Focused debugging workflow for a specific OpenClaw agent |
| deploy-task | Follow the OpenClaw governance model before deploying a change — envelope, dry run, explicit go-ahead |

## Personal infrastructure skills

Skills that require the OpenClaw companion system: [`personal-infrastructure/`](personal-infrastructure/README.md).
These are included as examples, not portable tools.
